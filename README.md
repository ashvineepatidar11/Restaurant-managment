1. Restaurant Management System (Beginner Python Project)

  A Restaurant Management System built using basic Python concepts.
  This project allows users to view a menu, place orders, and receive a final bill.
  It is designed for students and beginners who want to practice Python fundamentals such as dictionaries, loops, conditionals, and user input.


2. Features
  Displays a predefined restaurant menu with item names and prices
  Allows customers to order multiple items
  Validates user input (checks if item exists in the menu)
  Generates a clean bill receipt
  Calculates total amount automatically

3. How It Works
  A dictionary named menu stores food items and their prices.
  The program displays the menu to the user.
  The user enters item names to add them to their order.
  If the item exists, it is added to the order list.
  When the user types "done", the ordering process stops.
  The program prints a bill showing each item and its price.
  The total amount is calculated and displayed.

4. Tech Stack:
  Python (core language)
